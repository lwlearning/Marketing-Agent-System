# 电商营销策略知识库

## 用户分层策略与运营方法

### 高价值忠诚用户
- **定义**：最近30天内有购买，购买次数≥3次，总消费≥200美元
- **核心需求**：尊贵感、个性化服务、优先体验
- **营销策略**：
  - 专属折扣（9折及以上）
  - 免费配送服务
  - 新品优先购买权
  - 生日/节日专属礼物
  - VIP客服通道
- **触达渠道**：短信 + APP Push + 邮件
- **最佳触达时间**：工作日晚上8-10点
- **话术风格**：尊贵、专属、个性化

### 潜力增长用户
- **定义**：最近60天内有购买，购买次数1-2次，总消费100-200美元
- **核心需求**：性价比、优质商品、良好体验
- **营销策略**：
  - 满减优惠券（满100减20）
  - 第二件折扣
  - 相关商品推荐
  - 会员积分加倍
- **触达渠道**：APP Push + 短信
- **最佳触达时间**：周末下午2-6点
- **话术风格**：实惠、性价比、推荐

### 新用户
- **定义**：最近90天内首次购买，购买次数1次
- **核心需求**：引导、福利、信任
- **营销策略**：
  - 首单立减
  - 新人专享价
  - 新手指南
  - 首次复购优惠
- **触达渠道**：APP Push + 弹窗
- **最佳触达时间**：下单后7天内
- **话术风格**：欢迎、福利、引导

### 一般价值用户
- **定义**：最近180天内有购买，总消费50-100美元
- **核心需求**：便捷、优惠、实用
- **营销策略**：
  - 小额满减券（满50减10）
  - 限时折扣
  - 热销商品推荐
- **触达渠道**：APP Push
- **最佳触达时间**：工作日中午12-2点
- **话术风格**：优惠、便捷

### 流失风险用户
- **定义**：最近180天以上没有购买
- **核心需求**：关怀、召回、惊喜
- **营销策略**：
  - 回归专属优惠券（5折及以上）
  - 免运费
  - 个性化关怀信息
  - 新功能/新品通知
- **触达渠道**：短信 + 邮件
- **最佳触达时间**：流失后30天、60天、90天
- **话术风格**：关怀、召回、惊喜
