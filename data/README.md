# 数据集下载说明

本项目使用 **阿里云天池 - 淘宝用户行为数据集**：
- 下载链接：https://tianchi.aliyun.com/dataset/216549
- 文件名：`UserBehavior.csv`
- 下载后请放入此目录 (`data/`)

## 数据集简介
- 规模：约10万条用户行为记录
- 字段：
  - user_id: 用户ID
  - item_id: 商品ID
  - category_id: 商品类别ID
  - behavior_type: 行为类型 (pv=浏览, buy=购买, cart=加购, fav=收藏)
  - timestamp: 时间戳
